# SADAR Website

Professional static website for the SADAR project.

## Local preview

From the repository root:

```bash
python -m http.server 8080 -d website
```

Then open:

```text
http://localhost:8080
```

## Deployment

The site is static and can be deployed to GitHub Pages, Netlify, Vercel, Cloudflare Pages, or any basic web server.

Recommended GitHub Pages source:

- Folder: `website/`
- Entry file: `website/index.html`
